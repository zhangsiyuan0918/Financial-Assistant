# 财务分析仪表盘

个人记账数据清洗、支出预测、资产分析的 Web 工具。

---

## 数据清洗（4轮）

| 轮次 | 内容 | 产出 |
|------|------|------|
| R1 | 格式标准化（金额/日期/去emoji） | `记账数据_清洗_v1.csv` |
| R2 | 转账过滤 / AA退款修正 / 外币换算 | `记账数据_清洗_v2.csv` |
| R3 | 分类归一化（13类支出+6类收入）/ 节假日标记 / 时间特征 | `记账数据_清洗_v3.csv` |
| R4 | 异常值标记 / 生活阶段划分（pre_car/post_car）/ 资本支出标记 | `记账数据_清洗_v4.csv` |

- **数据规模**：7,195 条记录，2022.07 ~ 2026.07，42 个完整月份
- **清洗原则**：原始数据永不修改，每轮仅新增字段，支持完整回滚

## 项目结构

```
finance-dashboard/
├── backend/
│   ├── app.py                 Flask API 服务
│   ├── config.py              配置（资产/收入/车贷参数）
│   ├── requirements.txt
│   ├── utils/
│   │   └── data_loader.py     数据加载与业务逻辑
│   └── data/                  清洗数据 + 预测结果
├── frontend/
│   ├── src/
│   │   ├── main.js            Vue3 入口
│   │   ├── App.vue            布局（侧边栏导航）
│   │   ├── router/index.js    路由
│   │   ├── api/index.js       API 封装
│   │   └── views/
│   │       ├── Overview.vue    总览（净资产、资产饼图、收支趋势）
│   │       ├── Spending.vue    支出分析（分类饼图、明细下钻）
│   │       ├── Forecast.vue    支出预测（12月预测、置信区间）
│   │       └── Simulation.vue  情景模拟（参数调节、增长曲线）
│   ├── dist/                  已构建
│   └── vite.config.js
└── scripts/
    ├── prophet_forecast.py    Prophet 预测脚本
    └── clean_round4.py        数据清洗脚本
```

---

## 启动方式

```powershell
# 后端
cd backend
python app.py
# → http://localhost:5000

# 前端开发模式（另一个终端，热更新）
cd frontend
npm run dev
# → http://localhost:3000
```
