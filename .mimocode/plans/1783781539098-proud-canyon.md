# AI 财务助手 — 重新设计

## 产品定位转变

| | 之前 | 现在 |
|--|------|------|
| 定位 | 财务仪表板（看数据） | AI 财务助手（对话 + 建议） |
| 核心 | 图表、表格、多页面 | AI 对话、智能分析、主动推送 |
| 用户行为 | 用户主动翻页面看数据 | AI 主动告知 + 用户提问 |
| 竞品对标 | 记账软件（随手记、MoneyWiz） | AI 助手（ChatGPT + 个人数据） |

## 核心原则

1. **UI 极简** — 只需要：记账入口 + AI 对话窗口
2. **AI 是产品** — 不是装饰，是用户与数据交互的唯一方式
3. **后台自动化** — 数据聚合、分析、洞察生成都在后台完成
4. **主动推送** — 不等用户来问，AI 主动告知重要事项

## 二、AI 能力设计

### 2.1 对话能力（已有，增强）

当前 query_engine 已支持 16 个意图。增强：

| 能力 | 当前状态 | 增强方向 |
|------|---------|---------|
| 查询类 | ✅ 支出/收入/预算/持仓 | 增加跨维度组合查询 |
| 分析类 | ⚠️ 只有基础统计 | 增加趋势分析、同比环比 |
| 建议类 | ❌ 无 | 新增：基于数据的具体建议 |
| 预测类 | ❌ 无 | 新增：按当前速度预测月末 |

### 2.2 主动推送（新增）

AI 在以下时机主动推送洞察：

| 时机 | 推送内容 |
|------|---------|
| **记账后** | "这笔 ¥11,742 车辆支出高于月均 ¥400，建议关注预算" |
| **每日定时** | "今日支出 ¥280，本月累计 ¥12,935，按速度月底 ¥15,200" |
| **每周定时** | 周报：本周消费趋势、分类变化、异常检测 |
| **预算临界** | "餐饮预算已用 80%，剩余 ¥400，本月还剩 18 天" |
| **大额交易** | "检测到 ¥11,742 大额支出，是否需要调整预算？" |

### 3.4 定时任务

```python
# app.py 中
import threading
from datetime import datetime

def scheduled_tasks():
    """定时任务调度器"""
    while True:
        now = datetime.now()
        # 每天 23:00 生成日报
        if now.hour == 23 and now.minute == 0:
            generate_daily_insight()
        # 每周日 23:00 生成周报
        if now.weekday() == 6 and now.hour == 23 and now.minute == 0:
            generate_weekly_report()
        time.sleep(60)  # 每分钟检查一次

# 启动时
threading.Thread(target=scheduled_tasks, daemon=True).start()
```

## 七、验证方案

1. **记账流程**：在 AI 助手页面记一笔账 → AI 立即给出分析反馈
2. **对话查询**：问"我这个月花了多少钱" → AI 准确回答并给出建议
3. **主动推送**：记一笔大额交易 → AI 主动弹出提醒
4. **日报生成**：等待定时任务（或手动触发）→ AI 对话区域显示今日洞察
5. **设置**：配置 LLM → 对话正常工作
